    Condicional "Si"

    El condicional "Si" permite ejecutar un bloque de código solo si una condición específica es verdadera. En Flowgorithm, esta estructura se representa mediante un rombo, que evalúa la condición y decide el flujo del algoritmo.

    • Condición Verdadera: Si la condición se cumple (es verdadera), se ejecuta el bloque de código dentro del "Si".
    • Condición Falsa: Si la condición no se cumple (es falsa), se puede ejecutar un bloque de código alternativo usando "Sino".

    Ejemplo de Condicional "Si"

    Supongamos que tienes una variable, edad, y quieres comprobar si es mayor o igual a 18 para determinar si la persona es mayor de edad:

    
edad = 20

    El condicional "Si" en Flowgorithm podría ser:

    • Si (edad >= 18) entonces: La persona es mayor de edad.
    • Sino: La persona es menor de edad.

    Pasos para usar el condicional "Si" en Flowgorithm

    1. Declarar la Variable:
      • En la sección de Variables, crea una variable, edad.
      • Selecciona el tipo Integer (entero) para esta variable.
    2. Asignar un Valor a la Variable:
      • Usa una Sentencia de Asignación para establecer el valor de la variable. Por ejemplo, asigna 20 a edad.
    3. Agregar el Condicional "Si":
      • Usa el símbolo de rombo para agregar un condicional "Si" en Flowgorithm.
      • Especifica la condición, por ejemplo, edad >= 18.
      • Agrega los bloques de código dentro de las ramas "Si" y "Sino".
