    Ejercicio: Almacenar y Mostrar los Primeros Números Pares

    En este ejercicio, aprenderás a almacenar y mostrar los primeros 5 números pares utilizando una matriz en Flowgorithm. Sigue los pasos a continuación para completar el ejercicio:

    1. Crear un Nuevo Diagrama: Crea un nuevo diagrama de flujo.
    2. Declarar la Matriz: Declara una matriz llamada numeros de tamaño 5 para almacenar los números pares. La declaración debe ser algo así: Entero numeros[5].
    3. Declaramos una variable llamada contador y le asignamos el valor de 0 y una llamada I con la que ejecutaremos el bucle
    4. Inicializar la Matriz: Utiliza un bucle For para asignar los primeros 5 números pares a la matriz. Por ejemplo:
