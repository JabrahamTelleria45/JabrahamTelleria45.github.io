    Concepto de Variables String o cadena

    Las variables de tipo string se utilizan para almacenar cadenas de texto. Estas cadenas pueden incluir letras, números y caracteres especiales. Las variables string son útiles para almacenar información como nombres, apellidos, direcciones, descripciones y contraseñas.

    Ejemplo de Variable String:

    • nombre = "Juan"
    • apellido = "Pérez"
    • direccion = "Calle 123, Ciudad"
    • descripcion = "Este es un ejemplo de descripción."
    • contrasena = "miContraseña123"

    Nota: Las variables string siempre deben estar entre comillas, de lo contrario si declaras una variable cadena y la asignas tipo: nombre = Juan te marcará error ya que necesitan ir entre comillas.

    En Flowgorithm, para declarar y usar variables de tipo string, debes seguir estos pasos:

    Declaración de Variables String en Flowgorithm

    1. Declarar la Variable:
      • En la sección de Variables, crea una nueva variable. Por ejemplo, nombre.
      • Selecciona el tipo String(Cadena) para esta variable.
    2. Asignar un Valor a la Variable:
      • Usa una Sentencia de Asignación para establecer el valor de la variable. En este caso, asigna "Juan" a nombre.
    3. Salida de Datos:
      • Usa la Sentencia de Salida para mostrar el valor de la variable nombre.

    Ejemplos en Flowgorithm:

    
nombre = "Juan"
apellido = "Pérez"
escribir("Nombre Completo: " + nombre + " " + apellido)

    Ahi puedes observar que en la expresión de salida tengo una concatenación eso quiere decir que estoy uniendo las 2 variables

    Es decir: si yo pusiera un + me marcaria error porque estoy uniendo variables para escribirla en una salida no para hacer una operación por eso se utiliza el: &

    Conocerás más sobre operadores lógicos en otras clases

    Operaciones con Variables String

    En Flowgorithm, puedes realizar varias operaciones con variables string, como concatenación. A continuación, te muestro algunos ejemplos:

    Ejemplos:

    
saludo = "Hola, " & nombre
mensaje = "Tu apellido es " + apellido (Error ya que en vez de + sería &)
    Puedes practicarlo como tarea :3

    Recuerda siempre declarar el tipo correcto de variable dependiendo del tipo de dato que vas a almacenar y las operaciones que vas a realizar.
