    Concepto de Variables Enteras

    Recuerdas que decíamos sobre las variables en nuestra clase anterior? Que en programación, una variable es un espacio en memoria que se utiliza para almacenar un dato que puede tanto cambiar o permanecer durante la ejecución del programa. En Flowgorithm o cualquier otro lenguaje, las variables enteras se usan para almacenar números enteros (sin decimales).

    Ejemplo de Variable Entera:

    • fechaNacimiento = 15
    • notaAlumno = 86

    Nota: Si intentas escribir una variable entera de esta forma notaAlumno = "86", tendrás un error ya que entre comillas solo se ponen las variables de tipo string que veremos en siguientes capítulos.

    Como podemos ver nos marca un error al ser una variable entera no debe ser escrita ente comillas "" siempre recuerdalo.

    En estas variables también puedes almacenar sumas ya que siempre serán de tipo entero. Supongamos que quieres almacenar el resultado de una suma. Puedes declarar una variable y asignarle el valor de la suma de dos números:

    suma = 5 + 2

    En Flowgorithm, para hacer esto, seguirías estos pasos:

    1. Declarar la Variable:
      • En la sección de Variables, crea una nueva variable. Por ejemplo, suma.
      • Selecciona el tipo Integer (entero) para esta variable.
    2. Asignar un Valor a la Variable:
      • Usa una Sentencia de Asignación para establecer el valor de la variable. En este caso, asigna 7 a suma usando la expresión 5 + 2.

    Sobrecarga de Variables

    En Flowgorithm, puedes cambiar el valor de una variable en cualquier momento. Si tienes una variable edadAna y quieres actualizar su valor, simplemente puedes usar una sentencia de asignación para hacerlo.

    Ejemplo:

    edadAna = 23

    Después, si necesitas actualizar el valor de edadAna a 25, puedes hacer:

    edadAna = 25

    Recuerda que al declarar una variable, puedes cambiar su valor en cualquier momento ya sea creando durante la ejecución del algoritmo.

    Escribir Variables en camelCase

    Siempre recuerda que es una buena práctica usar el estilo camelCase para nombrar tus variables. Esto significa que la primera palabra comienza en minúscula y cada palabra subsiguiente comienza con una letra mayúscula, te dejo los ejemplos nuevamente.

    Ejemplos de Nombres de Variables en camelCase:

    • edadAna
    • numeroDeEstudiantes
    • sumaTotal
