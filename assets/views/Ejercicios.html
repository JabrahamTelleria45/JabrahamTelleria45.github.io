    Ejercicios Básicos

    A continuación, encontrarás una lista de ejercicios básicos para cada uno de los temas cubiertos en nuestras clases.

    Variables

    • Declarar una variable y asignarle un valor.
    • Intercambiar los valores de dos variables.
    • Convertir una variable de tipo entero a tipo decimal.
    • Suma de dos variables y mostrar el resultado.
    • Restar dos variables y mostrar el resultado.
    • Multiplicar dos variables y mostrar el resultado.
    • Dividir dos variables y mostrar el resultado.
    • Mostrar el tipo de una variable.

    Tipos de Datos

    • Declarar una variable de tipo entero y asignarle un valor.
    • Declarar una variable de tipo decimal y asignarle un valor.
    • Declarar una variable de tipo cadena y asignarle un valor.
    • Convertir una variable de tipo entero a tipo cadena.
    • Convertir una variable de tipo cadena a tipo entero.
    • Mostrar el valor de una variable de tipo booleano.
    • Realizar una operación aritmética con variables de diferentes tipos.
    • Concatenar dos variables de tipo cadena.

    Operadores Lógicos y Aritméticos

    • Utilizar el operador AND en una expresión lógica.
    • Utilizar el operador OR en una expresión lógica.
    • Utilizar el operador NOT en una expresión lógica.
    • Realizar una suma de dos números.
    • Realizar una resta de dos números.
    • Realizar una multiplicación de dos números.
    • Realizar una división de dos números.
    • Utilizar el operador módulo para obtener el residuo de una división.

    Estructuras de Control

    • Determinar si una persona es mayor de edad o no.
    • Por medio de entrada introducir una hora, si la hora es arriba de las 17 mostrar que ya es noche, si es arriba de las 12 que es de tarde, y si es antes de las 12 que es de mañana.
    • Determinar si un estudiante tiene derecho o no a beca, tomando en cuenta que el porcentaje de becas es arriba del 90%
    • Implementar un bucle WHILE para repetir una acción mientras el numero sea menor a 20.
    • Implementar un bucle DO que se ejecute hasta que el usuario ingrese un numero par
    • Implementar un bucle FOR para tener una lista del numero al 100 imprimiendo solo los numeros pares

    Matrices

    • Declarar una matriz y asignarle valores.
    • Acceder a un elemento específico de una matriz.
    • Modificar un elemento específico de una matriz.
    • Recorrer una matriz utilizando un bucle FOR.
    • Calcular la suma de todos los elementos de una matriz.
    • Encontrar el valor máximo en una matriz.
    • Encontrar el valor mínimo en una matriz.
    • Ordenar los elementos de una matriz en orden ascendente.
