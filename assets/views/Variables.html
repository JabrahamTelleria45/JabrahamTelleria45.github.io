    Variables en Programación

    En programación, una variable es un espacio de almacenamiento con un nombre asociado que puede contener un valor. Las variables permiten a los programas almacenar y manipular datos de manera eficiente. El nombre de la variable se utiliza para referirse al valor almacenado en ese espacio de almacenamiento.

    • edadUsuario
    • nombreCompleto
    • marcaAuto

    Estas convenciones ayudan a distinguir rápidamente los nombres de variables y a entender su propósito dentro del código.

    Tipos de Variables en Flowgorithm

    Flowgorithm es una herramienta educativa diseñada para enseñar los conceptos básicos de la programación a través de diagramas de flujo. Flowgorithm admite varios tipos de variables básicos que se utilizan comúnmente en la programación. Aquí están los principales tipos de variables que puedes utilizar en Flowgorithm:

    • Integer (Entero): Almacena números enteros, es decir, sin decimales. Ejemplo: int edad = 20
    • Real (Real): Almacena números con decimales. Ejemplo: real altura = 1.75
    • Array (Arreglo): Almacena múltiples valores del mismo tipo en una estructura indexada. Ejemplo: int[] numeros = {1, 2, 3, 4, 5}
    • Boolean (Booleano): Almacena valores de verdadero o falso. Ejemplo: boolean esMayorDeEdad = true
    • String (Cadena de caracteres): Almacena secuencias de caracteres, como palabras o frases. Ejemplo: string nombre = "Ana"

    Declaración y Asignación de Variables en Flowgorithm

    Declaración: Es el proceso de crear una variable y definir su tipo de dato. Ejemplo en Flowgorithm: int edad

    Asignación: Es el proceso de dar un valor a una variable. Ejemplo en Flowgorithm: edad = 20

    Variables Combinadas

    En programación, a veces es necesario realizar operaciones con variables de diferentes tipos. A esto se le llama "variables combinadas". Flowgorithm permite ciertas operaciones entre variables de tipos diferentes, siempre que las operaciones sean lógicas y compatibles.

    Ejemplo de combinación de variables:

    Supongamos que queremos calcular si una persona es mayor de edad. Para ello, necesitamos combinar variables de tipo entero y booleano.

    
int edad = 20
int edadMinima = 18
boolean esMayorDeEdad = edad >= edadMinima

    En este ejemplo, edad y edadMinima son variables enteras, y la expresión edad >= edadMinima es una comparación que devuelve un valor booleano (true o false), que se almacena en esMayorDeEdad.

    Ejercicio Práctico

    Vamos a realizar un ejercicio simple para reforzar el concepto de variables.

    Ejercicio: Cálculo de Área de un Rectángulo

    En este ejercicio, calcularemos el área de un rectángulo. Para ello, seguiremos los siguientes pasos:

    • Declara dos variables enteras: ancho y alto.
    • Asigna los valores 5 y 10 a las variables ancho y alto, respectivamente.
    • Declara una tercera variable entera llamada area.
    • Calcula el área multiplicando ancho por alto y asigna el resultado a la variable area.
    • Muestra el resultado del área en la consola.

    Diagrama de flujo en Flowgorithm:

    1. Declara las variables: ancho, alto y area.
    2. Asigna los valores: ancho = 5 y alto = 10.
    3. Calcula el área: area = ancho * alto.
    4. Muestra el resultado del área.
