    Concepto de Variables Reales

    Las variables reales, también conocidas como variables de punto flotante, se utilizan para almacenar números que tienen decimales. A diferencia de las variables enteras que solo pueden contener números sin decimales, las variables reales pueden contener tanto números enteros como fracciones.

    Ejemplo de Variable Real:

    • temperatura = 36.5
    • precioProducto = 19.99

    Nota: A diferencia de las variables enteras, las variables reales deben contener un punto decimal. Si intentas escribir una variable real sin el punto decimal, se considerará como una variable entera.

    En estas variables también puedes almacenar operaciones aritméticas que resulten en valores con decimales. Por ejemplo, si quieres almacenar el resultado de una división que no da un número entero, puedes usar una variable real:

    resultado = 5 / 2

    En Flowgorithm, para hacer esto, seguirías estos pasos:

    • Declara una variable llamada num1 (Esto es de tu preferencia, solo es un identificador) y una llamada num2.
    • Asignales el valor a cada una.
    • Luego en una variable llamada resultado almacena el resultado y asignale el valor dividiendo cada uno de los números.

    Operaciones con Variables Reales

    En Flowgorithm, puedes realizar varias operaciones aritméticas con variables reales, como suma, resta, multiplicación y división. A continuación, te muestro algunos ejemplos:

    Ejemplos:

    
  suma = 10.5 + 2.3
  resta = 5.8 - 1.2
  multiplicacion = 3.4 * 2.0
  division = 7.2 / 2.4
    Puedes practicarlo como tarea :3

    Como puedes ver, las operaciones con variables reales son similares a las operaciones con variables enteras, pero el resultado puede incluir decimales.

    Recuerda siempre declarar el tipo correcto de variable dependiendo del tipo de dato que vas a almacenar y las operaciones que vas a realizar.
